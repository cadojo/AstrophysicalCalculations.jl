AstrophysicalCalculations.jl

Common equations within modern astrophysics!

Overview

This package provides equations as they're written in Caroll and Ostlie's An Introduction to Modern Astrophysics. For now, all of the documentation exists within each function or constant's docstrings. Check the sidebar for functions and constants grouped into modules by their underlying assumptions and content.

Installation

This package is not-yet registered.

pkg> add https://github.com/cadojo/AstrophysicalCalculations.jl
